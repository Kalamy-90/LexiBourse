# LexiBourse — Design Ideas

## Approches envisagées

### 1. Dark Finance Premium (p=0.07)
Inspiré du site Daily Question, fond très sombre (#070812), accents or/ambre, typographie Playfair Display + Jakarta Sans. Cartes glassmorphism, gradients subtils, animations Framer Motion.

### 2. Trading Terminal (p=0.04)
Esthétique terminal de trading : fond charbon, vert néon (#00FF88), police monospace, grilles de données, effets scanlines.

### 3. Nuit Boursière Élégante (p=0.06)
Bleu nuit profond (identique à la bannière #0D1B3E), accents vert émeraude (#22C55E) et or, style magazine financier haut de gamme.

---

## Approche choisie : **Dark Finance Premium** (inspiré Daily Question)

Cette approche reprend fidèlement l'esthétique premium du site Daily Question tout en l'adaptant à l'univers financier de LexiBourse.

### Design Movement
Dark Luxury Finance — premium fintech avec influences editorial

### Core Principles
1. Fond très sombre quasi-noir avec gradients subtils violets/ambrés
2. Accents or/ambre (#F4C974) pour les éléments clés, vert (#22C55E) pour les données boursières
3. Glassmorphism sur les cartes (backdrop-blur, border white/10)
4. Typographie contrastée : Playfair Display (titres) + Plus Jakarta Sans (corps)

### Color Philosophy
- Background : #070812 (quasi-noir avec teinte bleue)
- Accent primaire : #F4C974 (or ambre — richesse, finance)
- Accent secondaire : #22C55E (vert bourse — hausse, croissance)
- Texte : slate-100 à slate-300
- Cards : white/4.5% avec backdrop-blur

### Layout Paradigm
Navigation flottante pill centrée (comme Daily Question), sections verticales avec max-w-6xl, grilles asymétriques pour les fonctionnalités, hero centré avec bannière pleine largeur.

### Signature Elements
1. Pill navigation flottante avec backdrop-blur
2. Cards "premium" avec gradient interne et hover doré
3. Eyebrow labels en uppercase tracking-widest ambre

### Interaction Philosophy
Transitions fluides 200-300ms, hover avec légère élévation (-translate-y-1), animations d'entrée en scroll avec Framer Motion.

### Animation
- Entrée sections : fade-up avec stagger 80ms
- Cards : hover scale légère + border amber
- Hero : fade-in séquentiel

### Typography System
- Display : Playfair Display (titres de section, hero)
- Body : Plus Jakarta Sans (navigation, paragraphes)
- Mono : JetBrains Mono (commandes bot)

### Brand Essence
LexiBourse — le marché boursier de vos conversations Discord. Pour les communautés qui veulent jouer avec les mots et l'argent virtuel.
Adjectifs : Dynamique, Intelligent, Ludique

### Brand Voice
"Transformez chaque mot en investissement" — "La bourse de votre serveur, en temps réel"
Pas de "Bienvenue sur notre site" ni "Commencez maintenant" génériques.

### Signature Brand Color
#F4C974 (or ambre) — la couleur des pièces, de la richesse, du succès boursier.
